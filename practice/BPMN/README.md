# BPMN-Диаграммы: Практика по моделированию бизнес-процессов

В этой папке собраны примеры диаграмм бизнес-процессов, выполненных мной в формате BPMN.  
Цель — отработать навыки визуального моделирования процессов, понять структуру и логику потоков.
