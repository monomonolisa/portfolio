# SQLite: Практические запросы и работа с базой данных

В этой папке собраны практические задания по работе с базой данных в **SQLite**. 
Здесь представлены запросы, написанные для анализа данных и извлечения бизнес-инсайтов на основе структурированной схемы.

## Содержимое папки

- [`SQLite.sql`](https://github.com/monomonolisa/portfolio/blob/23beb6e31ae3328d617e07f451cb956fb84a523c/practice/SQL/SQLite/SQLite.sql) — файл с SQL-запросами, включающий:
  - Сложные JOIN-соединения между таблицами
  - Подзапросы и вложенные SELECT
  - Агрегации с GROUP BY и HAVING
  - Расчётные поля, фильтрация и сортировка результатов

## Продемонстрированные навыки

- Чтение и интерпретация ER-диаграммы для понимания структуры данных.
- Написание сложных запросов для извлечения информации из нескольких связанных таблиц.
- Использование агрегирующих функций (`SUM`, `AVG`, `COUNT`) с группировкой и фильтрацией.
- Применение `HAVING` для фильтрации сгруппированных данных.
