# Практика по MySQL

> *Решённые задачи из курса [Интерактивный тренажер по SQL](https://stepik.org/63054)*

Эта папка содержит практические задания по MySQL, выполненные в рамках обучения на платформе Stepik. 
Цель — закрепить навыки работы с реляционными базами данных, написанием запросов, созданием таблиц, вставкой данных, 
использованием JOIN, подзапросов, группировок и агрегатных функций.

## Что я освоила

- Создание таблиц (`CREATE TABLE`)
- Вставка данных (`INSERT INTO`)
- Выборка данных (`SELECT`, `WHERE`, `ORDER BY`)
- Агрегатные функции (`COUNT`, `SUM`, `AVG`, `MIN`, `MAX`)
- Группировка (`GROUP BY`, `HAVING`)
- Подзапросы (`IN`, `EXISTS`, вложенные `SELECT`)
- Соединения таблиц (`INNER JOIN`, `LEFT JOIN`, `RIGHT JOIN`)
- Логические и текстовые операции в SQL
