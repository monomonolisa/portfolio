# Проект: Сервис назначения задач менеджерам

> **Цель**: Автоматизировать ручной процесс назначения и контроля задач, заменив Excel-файлы на систему в CRM.  
> **Автор**: Хайдарова А.Р. | **Версия**: v0.1 (черновик) | **Дата**: 08.10.2025

## Бизнес-цель

Повысить эффективность отработки клиентских задач менеджерами на **40% за 6 месяцев** за счёт автоматизации процесса назначения и контроля.

### Критерии успеха (KPI)
- **БЦ 1.1**: ≥90% задач взяты в работу в течение 1 дня.
- **БЦ 1.2**: ≥75% задач закрыты за 3 дня после контакта.
- **БЦ 1.3**: ↓20% просроченных задач ежемесячно.

## Как всё связано?

Этот проект — единая цепочка от бизнес-идеи до технической реализации:

1. **Бизнес-цели** → определяют, что нужно сделать.
2. **Use Case** → описывают, как пользователь будет взаимодействовать с системой.
3. **BPMN** → визуализирует процессы, описанные в Use Case.
4. **ER-диаграмма** → показывает, как данные будут храниться и связываться.
5. **UI-макет** → демонстрирует, как это будет выглядеть для пользователя.
